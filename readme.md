To operate YBF:

1. Install YBF's prerequisites:
	* Python 3.6
	* https://github.com/Rapptz/discord.py
		(follow "to install the development version" instructions)
2. Create the `./ybf/configs` folder and add your token and base config to it
	(See the provided examples.)
3.  Start YBF by double-clicking `start.bat` in the root directory. (Windows)
	or `reset.sh` (Linux)
